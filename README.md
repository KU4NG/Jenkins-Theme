##项目说明

闲来无事，发现 Jenkins 的默认主题实在看的有点腻了，便想要不自己写一套？于是有了这个项目。

##Jenkins任何使用主题？

*Jenkins 在安装之后需要安装插件：Simple Theme Plugin
*安装插件之后重启 Jenkins，然后在 [系统管理] --> [系统设置] --> [Theme] --> [Extra CSS]
*将我们的 style.css 文件中的内容粘贴到此处即可
